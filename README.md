**Features**
  3 Allocation Algorithms: First Fit, Best Fit, Worst Fit--
  Real-time Visualizations: Color-coded memory blocks with fragmentation overlay--
  Multiple Layouts: Single, Multiple, Fragmented, and Mixed partitions--
  Dynamic/Fixed Partitioning: Toggle between allocation modes--
  Analytics Dashboard: Fragmentation stats, memory utilization, comparison charts--
  Interactive Controls: Add/terminate processes, memory compaction, random generation--

**Tech Stack**
  Frontend: Vanilla JavaScript (ES6+), HTML5, CSS3--
  Visualization: Chart.js for analytics--
  No Dependencies: Runs entirely client-side--
  Responsive Design: Works on desktop & tablet--
